## How to Add Yourself

We use the [All Contributors](https://allcontributors.org/) bot to manage our contributors list.

To add yourself to the contributors list, comment on an issue or PR with:

```
@all-contributors please add @username for code, doc
```

Where `username` is your GitHub username and the contribution types can be:

- `code` - Code contributions
- `doc` - Documentation
- `bug` - Bug reports
- `ideas` - Ideas and planning
- `review` - Reviewing pull requests
- `test` - Testing
- `infra` - Infrastructure (CI/CD, build tools, etc.)
- `design` - Design contributions
- `maintenance` - Repository maintenance
- `translation` - Translation
- `question` - Answering questions
- `tutorial` - Tutorials

See the [emoji key](https://allcontributors.org/docs/en/emoji-key) for all contribution types.
