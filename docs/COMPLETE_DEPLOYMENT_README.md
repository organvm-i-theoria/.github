## ðŸ’¡ Quick Start

### Generate Your First Walkthrough

```bash
# Via GitHub CLI
gh workflow run generate-walkthrough.yml

# Via GitHub UI
1. Go to Actions tab
2. Select "Generate Video Walkthrough"
3. Click "Run workflow"
4. Wait 5-10 minutes
5. Review PR with video
```

### Deploy Your First Live App

```bash
# Via GitHub CLI
gh workflow run deploy-to-pages-live.yml \
  -f app_name="my-app" \
  -f app_type="react"

# Via GitHub UI
1. Go to Actions tab
2. Select "Phase 3: Deploy Live App to Pages"
3. Click "Run workflow"
4. Enter app name and type
5. Wait for deployment
6. Access at https://{org}.github.io/.github/apps/my-app
```

### Review Usage Metrics

```bash
# Check latest report
```
